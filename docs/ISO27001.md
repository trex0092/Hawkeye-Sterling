# ISO/IEC 27001:2022 Annex-A Controls Matrix — Hawkeye Sterling

Maps each Annex-A control to the implementing repo evidence. The
2022 revision consolidates the 2013 list into 93 controls across 4
themes: Organisational (37), People (8), Physical (14), Technological (34).

This is the **Statement of Applicability (SoA)** input — an external
ISO 27001 lead auditor uses it to scope Stage-1/Stage-2 fieldwork.
Certification requires an ISMS, internal audit, management review,
and Stage-2 audit by an accredited certification body.
