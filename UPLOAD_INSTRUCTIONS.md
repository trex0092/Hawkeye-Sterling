# How to Upload to GitHub (No Terminal Required)

Follow these steps in VS Code to push the standalone HTML to your repo.

---

## Step 1: Download This Folder

Download the `github_upload_ready` folder (click the download link above).

---

## Step 2: Extract and Copy

1. **Extract the zip** you just downloaded
2. **Open your local Hawkeye-Sterling folder** in File Explorer/Finder
3. **Drag the `Hawkeye Sterling v5 Index.html` file** from the extracted folder into your repo

---

## Step 3: Commit in VS Code

1. **Open VS Code** with your Hawkeye-Sterling folder
2. **Click the Source Control icon** in the left sidebar (looks like a branch/fork)
3. You'll see `Hawkeye Sterling v5 Index.html` listed under "Changes"
4. **Hover over the file** and click the **+** icon to stage it
5. **Type a commit message** in the box at the top:
   ```
   Add standalone Hawkeye Sterling v5 index page
   ```
6. **Click the ✓ Commit button** (above the message box)

---

## Step 4: Push to GitHub

1. **Click the ⋯ (three dots)** at the top of the Source Control panel
2. **Select "Push"**
3. Wait for the upload to complete

---

## Done!

Your standalone HTML is now on GitHub. You can:
- View it in your repo browser
- Share the raw GitHub URL
- Deploy it to GitHub Pages

## Optional: Deploy to GitHub Pages

If you want a live URL:

1. Go to your repo on GitHub.com
2. Click **Settings** → **Pages**
3. Under "Source", select your branch (probably `main`)
4. Save
5. GitHub will give you a URL like: `https://trex0092.github.io/Hawkeye-Sterling/`

The standalone HTML will work perfectly on GitHub Pages.
