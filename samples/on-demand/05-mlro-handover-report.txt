1. PURPOSE AND STANDING OF THIS DOCUMENT
-----------------------------------------------------------------------------

This document is a continuity snapshot prepared by the compliance
function to support the handover of the Money Laundering Reporting
Officer role at [Reporting Entity]. It is not a formal appointment
letter. The appointment of the incoming MLRO is a matter for the Board.
This document records the operational state of the firm's AML and CFT
programme at the point of handover so that the incoming MLRO can take
up the role with a clear understanding of what is in place, what is
outstanding, and what requires immediate attention.

4. FILING MODE CONFIGURATION AT HANDOVER
-----------------------------------------------------------------------------

All five goAML filing types are configured in manual mode:

Filing type                                    Mode
-------------------------------------------    ------
Suspicious Transaction Report (STR)            Manual
Suspicious Activity Report (SAR)               Manual
Dealers in Precious Metals and Stones
  Report (DPMSR)                               Manual
Partial Name Match Report (PNMR)              Manual
Funds Freeze Report (FFR)                      Manual

Manual mode means that the compliance function prepares draft filings
but does not submit any filing to the Financial Intelligence Unit
without the MLRO's express instruction. The incoming MLRO may change
the filing mode by editing the configuration file at
scripts/filing-mode.json.
